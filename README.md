# README 

## 開発言語 
- Ruby 3.0.1  
- Rails 6.0.3  
## 就業Termの技術 
-  device機能
-  Ajaxを使ったコメント機能
## カリキュラム外の技術 
-  SimpleCalendar
## 実行手順 
```bash
$ git clone git@github.com:ayaka2209/original_app.git    
$ cd original_app    
$ bundle install 
$ rails db:create && rails db:migrate 
$ rails s  
 ```
